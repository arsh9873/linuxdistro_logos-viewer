# linuxdistro_logos-viewer
a simple bash script using neofetch to display alot of distro logos in your terminal itself
<br>
<h3>REQUIRMENTS:</h3><br>
  ->neofetch:<br>
    install using:<br>
    $sudo apt install neofetch<br>
<h3>Running:<br></h3>
   $bash viewlogos.sh<br>
